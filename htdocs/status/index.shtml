OpenHPI :: Status

OpenHPI is an open source project, so is continually a work in progress. As OpenHPI 1.0 is a set of 56 APIs, which are fullfilled by both the OpenHPI infrastructure and OpenHPI plugins. We will be providing tracking of the status of the APIs based on the following key.

BrokenFunctionality hasn't been started yet, or is in a non operational state
Not DoneFunctionality is in progress. It probably doesn't do what it should, but it shouldn't crash the system.
Semi FunctionalFunctionality exists, but isn't guaranteed to work properly. Considered alpha level functionality.
FunctionalFunctionality exists, and should work. It has not be extensively tested, and should be considered beta quality
DoneFunctionality exists and has been tested aggressively. Considered GA quality
ImpossibleIt is impossible to implement this function. This may be the case for some functions on some hardware.
No ApplicableThis code isn't supposed to do this function. This is used for HPI calls that are entirely handled in the infrastructure.
No DataThere is no information yet on this status. Please check back later.
