The OpenHPI Project

Open Hardware Platform Interface

Status

OpenHPI is an open source project, so is continually a work in progress. HPI B.01.01 is a set of 79 APIs, which are fullfilled by both the OpenHPI infrastructure and OpenHPI plugins. We will be providing tracking of the status of the APIs based on the following key.

Legend

Broken Functionality hasn't been started yet, or is in a non operational state
Not Done Functionality is in progress. It probably doesn't do what it should, but it shouldn't crash the system.
Semi Functional Functionality exists, but isn't guaranteed to work properly. Considered alpha level functionality.
Functional Functionality exists, and should work. It has not be extensively tested, and should be considered beta quality
Done Functionality exists and has been tested aggressively. Considered GA quality
Impossible It is impossible to implement this function. This may be the case for some functions on some hardware.
No Applicable This code isn't supposed to do this function. This is used for HPI calls that are entirely handled in the infrastructure.
No DataThere is no information yet on this status. Please check back later.
